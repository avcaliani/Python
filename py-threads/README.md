This is an example of using multiple threads to download multiple files at same time.

## Quick Start

```bash
python3.9 -m venv .venv &&
  pip install wget==3.2 &&
  source .venv/bin/activate

# Single Thread
python single-file.py

# Thread Pool
python multiple-files.py
```
